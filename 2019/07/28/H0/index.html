计算机分类：

根据处理的对象划分，计算机分为模拟计算机、数字计算机、混合计算机

根据计算机用途划分，计算机可分为专用计算机、通用计算机

根据计算机的规模划分，巨型机、大型机、小型机、微型机、工作站

CAD 计算机辅助设计

CAM 计算机辅助制造

CAT 计算机辅助测试

CIMS 计算机集成制造系统

CEB(计算机辅助教育)包括 CAI （辅助教学）CMI（辅助管理）

计算机的发展趋势 1.巨型化 2.微型化

计算机的主要应用：1.科学计算 2.信息管理(数据处理) 3.过程控制

进制

数码： 1、2、3、4、A、B、C、I、II、III、IV

基数：数码的个数

位权

A—Z : 65-90 a—z : 97- 122 小写字母大 大20H（30）

位 比特 bit 计算机储存数据的最小的单位

字节 Byte B 1B = 8bit 储存信息的基本单位

1024b=1B

1024B= 1KB

1024KB=1M

1024M= 1GB

1024GB= 1TB

ASCII码 7位ASCII码称为标准ASCII码，用一个字节表示(8位)表示一个字符，并规定其最高为为0，实际只用到7位，因此可表示128个不同的字符 8位ASCII码称为扩展ASCII码

汉字交换码 1980《信息交换用汉字编码字符集基本集》(GB2312-80)简称国标码

汉字机内码 为了防止混淆 把国标码两个字节的最高为改为1 称为 机内码

汉字字形码 点阵码（点阵法）和矢量码（矢量法）

汉字输入码 流水码、音码（we平时用的）、形码(五笔）、音形结合码、

指令

指令是指示计算机执行某种操作的命令，它有一串二进制数码组成，包括操作码和地址码

计算机系统不同，指令系统也不同 常见的指令系统有 复杂指令系统(CISC)和精简指令系统(RISC)

计算机能够自动完成运算或处理过程的基础是“储存程序”工作原理，由冯*诺依曼提出，故称冯·诺依曼原理，其基本思想是储存程序与程序控制

计算机的工作过程 (1)取指令(2)分析指令(3)执行指令

冯·诺依曼提出的“储存程序” 工作原理决定了计算机硬件系统由五个基本组成部分组成，即运算器、控制器、储存器、输入设备、输出设备

输入设备的主要主要功能是，把原始数据和处理这些数据的程序转换为计算机能够识别的二进制代码

运算器由逻辑运算单元（ALU）和寄存器组成 可以完成 加减乘除四则运算以及 与、或、非、位移操作

控制器是指挥和控制计算机各个部件进行工作的“神经中枢”

控制器和运算器合称为中央处理器（CPU）

高速缓存（cache ）协调RAM和CPU

输出设备 ：显示器、打印机、音响、绘图仪、各种数/模转换器（D/A）

QQ(软件)包括多个功能(程序)

软件是指计算机运行所需要的 程序、数据、有关文档的总和

软件系统包括 系统软件【操作系统、数据库管理系统、系统支撑和服务程序、语言处理程序】和应用软件

CPU不能直接访问外部储存

BOIS 存放基本输入/输出硬件程序 系统配置信息 系统引导程序和自检程序的ROM芯片

CMOS 可读写的RAM芯片，存放时间 日期硬盘参数 计算机配置 配有电池

程序设计基础包括 算法【有穷性、确定性、可行性】和数据结构【线性表、堆栈、队列】

栈：FILO 先进后出 队： FIFO 先进先出
